Music Press Pro - Manage your store: Genre, Artist, Albums and support audio and video file.

== Description ==
Music Press Pro helps you create and manage your music store: Genre, Artists and Albums. You can create audio and video playlist in pages or widgets.

<h4>Features</h4>
<ul>
<li>Genre management: Name, Image and Description</li>
<li>Album management: Name, Image, Album type, Description, Publication Date.</li>
<li>Artist, Band management: Name, Image, Description, Birthday, Occupation, Instruments, Website.</li>
<li>Audio, Video song: Name, Lyric, Description, Artist, multiple albums, free song.</li>
<li>Shortcode to display album.</li>
<li>Compatible with Tablet, Smartphone.</li>
<li>Compatible with Visual Composer plugin.</li>
</ul>

== Installation ==

<h4>1. Automatic installation:</h4>

The simplest way to install is to click on 'Plugins' then 'Add new' and type 'Music Press Pro' in the search field.

<h4>2. Manual installation 1:</h4>
<ol>
<li>Login to your website and go to the Plugins.</li>
<li>Click the Add New button.</li>
<li>Under Install Plugins, click the Upload Plugin.</li>
<li>Select the plugin zip file (music-press-pro.zip) from your computer then click the Install Now button.</li>
<li>You should see a message stating that the plugin is installed successfully.</li>
<li>Click the Activate Plugin.</li>
</ol>

<h4>2. Manual installation 2:</h4>
<ol>
<li>You should have access to the server where WordPress is installed.</li>
<li>Upload the plugin zip file (music-press-pro.zip) up to your server path(/wp-content/plugins/) and unzip it here.</li>
<li>Login to your website and go to the Plugins section of your admin panel.</li>
<li>Look for "Music Press" and click Activate.</li>
</ol>

== Frequently Asked Questions ==

= How to use shortcode to display album? =

Please add this shortcode to text editor in post or page or widget: <br>
[music_press_pro_album album_id="25" autoplay="1" volume="show" repeat="show" poster="show"]
